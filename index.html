        Guide

        Instructions

        Get The Data Folder

        1. Enable Dev Mode in Content Manager if you have not already. Open About at the top-right, then rapidly click the version number text, such as Version: 0.8.2790.... When Content Manager asks whether to enable Developer Mode, choose Yes.
        2. Open the car in Content Manager. Go to Content at the top-right, open the car list, and select the car you want to modify.
        3. Unpack the physics data. Click Unpack data at the bottom of the car page. Content Manager will generate a data folder inside that car's folder.

        Use This Tool

        1. Import only the data folder when possible. This keeps the editor fast and avoids loading car models, skins, previews, and sounds.
        2. If you accidentally import the full car folder, that is okay. The tool scans paths, loads only data/**, data.acd, and small metadata, then ignores graphics and sound assets.
        3. Edit with the tabs. Use Basics for mass/fuel/steering, Engine for limiter/inertia/power curve, Drivetrain for gears/diff/clutch, Brakes for torque/bias, Suspension for geometry/springs/dampers, and Tyres for pressure/grip values.
        4. Use Diagnostics and Changes. Open the header status button to review preflight or validation warnings, and check Changes so you know what you are replacing.
        5. Download the data zip. Replace the car's loose data folder with the exported one only after backing up the original car.

        Troubleshooting

        1. No editable data found: the car probably still has only data.acd. Use Content Manager's Unpack data, then import again.
        2. Changes do not appear in-game: Assetto Corsa may still be using data.acd instead of loose data. Confirm the car has a loose data folder and Content Manager is using it.
        3. Online server mismatch: edited physics usually fail checksum unless the server and all clients use the exact same files.
        4. Missing LUT warnings: referenced curve files are absent. Re-unpack the car or restore the missing files before exporting.
        Back up the car first. Replacing the generated data folder or its files changes the car copy Assetto Corsa will use. Duplicate the car folder or save a backup before editing.

        This editor is intentionally data-only. It does not edit models, skins, previews, sounds, or packed data.acd files.
